# Overview

This release introduces the Screener Engine, completing the first
version of AlphaEdge AI's opportunity screening layer.

The Screener Engine accepts screened trading opportunities,
applies configurable filtering rules, and produces a collection
of opportunities ready for dashboards, alerts, AI explanations,
and future market scanning capabilities.
