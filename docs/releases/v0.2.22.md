# Overview

This release introduces the Portfolio Engine, providing
centralized portfolio state management for AlphaEdge AI.

The Portfolio Engine maintains portfolio information such
as positions and capital allocation while intentionally
separating state management from performance analytics.

This architecture provides a stable foundation for
Alerts, Dashboard, Broker Integration, Watchlists and
future Portfolio Analytics.
