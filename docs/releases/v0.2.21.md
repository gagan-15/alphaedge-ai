# Overview

This release introduces the AI Explanation Engine, enabling
AlphaEdge AI to transform deterministic trading evidence into
clear, structured, and reusable explanations.

The explanation engine is intentionally deterministic and
provider-independent, allowing future integration with
OpenAI, Gemini, Claude, or local language models without
changing the core business logic.

---

# New Features

## AI Explanation Engine

Implemented:

- Structured explanation generation
- BUY / SELL / HOLD / WAIT decisions
- Configurable explanation length
- Optional confidence score
- Standardized AIExplanationResult
