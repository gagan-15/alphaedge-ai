# Overview

This release introduces the Backtesting Engine, providing the
first historical strategy evaluation capability within
AlphaEdge AI.

The Backtesting Engine evaluates completed trade outcomes and
produces standardized performance metrics that will become the
foundation for future analytics, AI explanations, portfolio
analysis, and strategy optimization.
