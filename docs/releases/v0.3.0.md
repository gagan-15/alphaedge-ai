# AlphaEdge AI v0.3.0

## Release Date

18 July 2026

---

## Overview

This release marks AlphaEdge AI's transition from a domain engine library to a backend application.

The project now exposes REST APIs using FastAPI and provides interactive API documentation through Swagger UI.

---

## New Features

### FastAPI Foundation

- Added FastAPI application
- Added API router
- Added Health API
- Added Dashboard API
- Added Swagger documentation

### Dashboard Service

- Added DashboardService
- Connected DashboardService to DashboardEngine
- Established Service Layer architecture

### Dashboard API

- Added `/dashboard`
- Returns complete DashboardResult
- Serializes nested domain models correctly

### Health API

- Added `/health`
- Returns backend health status
