# Implemented APIs

## Health API

### Endpoint

```
GET /health/
```

### Purpose

Verify that the AlphaEdge AI backend is running.

### Response

```json
{
    "status": "healthy"
}
```

---

## Dashboard API

### Endpoint

```
GET /dashboard/
```

### Purpose

Returns the complete dashboard model.

This endpoint currently returns sample data generated through the Dashboard Service and Dashboard Engine.

Future versions will populate this response using live engine outputs.

### Response Structure

- Portfolio
- Alerts
- Market Scanner
- Backtest
- AI Explanation

Example

```json
{
    "portfolio": {
        "total_positions": 3,
        "invested_capital": 25000,
        "available_capital": 75000,
        "total_capital": 100000
    },
    "alerts": [
        {
            "title": "BUY",
            "message": "INFY BUY",
            "priority": "HIGH",
            "requires_action": true
        }
    ],
    "scanner": {
        "scanned_symbols": 100,
        "screener_result": {
            "opportunities": []
        }
    },
    "backtest": {
        "total_trades": 100,
        "winning_trades": 70,
        "losing_trades": 30,
        "win_rate": 70.0
    },
    "ai_explanation": {
        "decision": "BUY",
        "reasons": [
            "Weekly Demand Zone"
        ],
        "confidence_score": 92.0,
        "summary": "Weekly Demand Zone"
    }
}
```
