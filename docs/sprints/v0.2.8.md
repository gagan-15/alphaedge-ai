# Sprint 2.30 — Change of Character (CHoCH) Engine

## Status

✅ Completed

## Overview

Sprint 2.30 introduces the Change of Character (CHoCH) Engine, enabling AlphaEdge AI to detect the first confirmed structural shift after a sustained bullish or bearish market trend.

The engine combines trend resolution with Break of Structure (BOS) detection to identify potential market reversals while maintaining strict validation, configurable confirmation logic, and full unit test coverage.

---

## Features

### CHoCH Engine

- Detect bullish Change of Character events
- Detect bearish Change of Character events
- Uses confirmed BOS events
- Integrates with Structure Trend Resolver
- Supports configurable confirmation source
- Supports configurable break buffers
- Supports equal-break configuration
- Returns complete CHoCH result object
